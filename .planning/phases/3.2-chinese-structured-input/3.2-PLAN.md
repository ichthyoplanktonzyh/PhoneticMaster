# Phase 3.2: Chinese Structured Input

## 目标

将中文拼音拼写模式从单纯逐字符/逐 token 追加，升级为声母 → 韵母 → 声调的结构化输入辅助，同时保留自由输入兜底。

## 产品判断

声母 → 韵母 → 声调是中文教学中常见的拼音拆解方式，适合作为输入顺序提示；但真实输入习惯通常是直接键入完整拼音，所以本阶段不取消自由输入，也不新增重型组合面板。
