## 浏览器冒烟

| 场景 | 操作 | 期望 |
|---|---|---|
| 中文拼写结构化输入 | 选择中文 → 拼写 → 打开键盘 | 显示声母、韵母、声调三段 |
| 单音节提交 | 选 `n` + `i` + `3`，点击添加 | 输入框出现 `ni3` |
| 多音节追加 | 再选 `h` + `ao` + `3`，点击添加 | 输入框出现 `ni3 hao3` |
| 自由输入兜底 | 直接在输入框键入 `ni3 hao3` | 仍可提交并判定 |
| 英语键盘不受影响 | 切到 English | 仍显示原 IPA keypad |

## 产品同步点

声母 → 韵母 → 声调适合中文课堂/教材式学习，但真实输入习惯未必如此。后续应和目标用户确认：

- 默认展示结构化面板是否比普通拼音键盘更直观
- “声母/韵母/声调”的术语是否需要更简化
- 是否需要一键切换“结构化 / 普通键盘”
