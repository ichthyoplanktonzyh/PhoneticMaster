# Phase 3.2 上下文

## 来源

Phase 3.1 已完成最小对立体 A/B 听辨。用户完成 QA 后反馈：

- Web Speech API 当前“勉强可用”，其他功能暂无问题。
- 后期最小对立体应替换为标准音频或更高质量 TTS。
- 下一步启动 Phase 3.2，并确认“声母 → 韵母 → 声调”是否符合中文学习者习惯。

## 关键决策

- 结构化输入采用“教学辅助”定位，不强制替代自由输入。
- 中文拼写模式保留文本输入框；用户仍可直接输入 `ni3 hao3`。
- 屏幕输入辅助按声母、韵母、声调组合音节，提交到同一个 `userInput` 字符串。
- 输出应尽量符合常见 tone-number 拼音正字法，例如 `qu4`、`xue2`、`yue4`；内部解析仍由 `parsePinyin` 归一。

## 约束

- 不改变 `zhWordBank.ts` 的 canonical pronunciation。
- 不改 `zhJudge` 的核心判定语义。
- 不引入外部拼音库。
- 不影响英语 IPA 键盘。
