# Phase 5.4 上下文

## 来源

Phase 5.1-5.3 建立课程模型、试点内容和可信媒体。Phase 5.4 才将它们呈现为可选 LearningPath，并证明课程可以复用现有训练能力而不把平台改造成强制线性课程。

## 关键决策

- “自由训练”“课程学习”“智能教练”是平等入口。
- 课程运行器根据 TrainingActivity reference 调用稳定的训练应用接口和现有视图。
- 课程可以增加目标说明、前后导航和达标解释，但不得复制出题、判定、答案记录和计分。
- CourseProgress 单向引用训练 session/result；TrainingSession 不反向依赖 Course。
- 自由训练产生的兼容 TrainingEvidence 可以计入课程目标。
- 所有顺序均为软引导，不锁课、不限制主题访问。

## 约束

- 没有课程的语言继续正常训练。
- 本地进度存储失败只影响课程进度展示，不影响内容和训练。
- 浏览内容不能被伪装成正确率或掌握度。
