# Phase 3.1: Targeted Minimal Pairs

## 目标

在独立训练器成立后，加入最小对立体听辨训练，让用户可以针对具体混淆音进行 A/B 选择练习。

## 任务

- [x] 3.1.1 定义 minimal pair 训练数据结构，避免只依赖 `"ship/sheep"` 字符串
- [x] 3.1.2 从 L1L2Difficulty.minimalPairs 或独立数据源生成练习题
- [x] 3.1.3 新增最小对立体训练视图：播放目标音，用户选择听到的词
- [x] 3.1.4 支持手动从音素/声调主题进入 minimal pair 训练
- [x] 3.1.5 统计本轮 minimal pair 正确率
- [x] 3.1.6 标注 Web Speech API 发音质量限制，为未来标准音频预留字段

## 退出条件

- [x] 用户可以选择某个音素进入 A/B 听辨
- [x] 每题有明确播放对象和两个或多个候选项
- [x] 完成本轮后显示正确率和错题
- [x] minimal pair 数据可被英语和中文复用或扩展
- [x] `npx tsc --noEmit` 通过
- [x] `npm run build` 通过
