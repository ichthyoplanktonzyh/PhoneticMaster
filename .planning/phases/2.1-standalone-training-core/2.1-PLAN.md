# Phase 2.1: Standalone Training Core

## 目标

让用户无需 L1 诊断、无需账号、无需后端，即可选择目标语言和训练类型并完成基础训练流程。

## 任务

- [x] 2.1.1 调整首次入口：L1 选择变为可选，L2 是进入训练的最小必要选择
- [x] 2.1.2 设计训练器主页/入口状态：目标语言、训练类型、难度、主题
- [x] 2.1.3 保留并整理听音拼写模式，确保英语 IPA 与中文拼音都可独立训练
- [x] 2.1.4 保留并整理看词听音模式，作为不需要输入的基础训练方式
- [x] 2.1.5 抽离训练设置与抽题逻辑，减少 App.tsx 顶层职责
- [x] 2.1.6 确认无 L1 / 无 difficultyMap 时不会阻塞训练

## 退出条件

- [x] 首次访问时，用户可以不选择 L1 而进入训练
- [x] 用户可以选择 L2、训练类型、难度和主题
- [x] 听音拼写和看词听音都具备完整进度导航
- [x] `npx tsc --noEmit` 通过
- [x] `npm run build` 通过

## 影响文件

- `src/App.tsx`

- `src/components/PhoneticKeypad.tsx`

## 约束

- 不新增后端依赖
- 不引入账号体系
- 不把 L1-aware 推荐作为进入训练的前置条件
