# Phase 4.1: Local Personalization

## 目标

用本地训练历史计算音素掌握度，并结合 L1-aware 难度映射生成下一步训练建议。

## 任务

- [x] 4.1.1 设计 MasteryRecord：按 L2、训练主题、音素/声调聚合正确率
- [x] 4.1.2 从训练会话结果更新本地掌握度
- [x] 4.1.3 将 L1 difficulty level 与用户正确率融合成推荐分
- [x] 4.1.4 在训练完成后展示“下一步建议”
- [x] 4.1.5 调整 SmartRecommend：从前置面板变为可选教练层
- [x] 4.1.6 提供清除个性化数据入口

## 退出条件

- [x] 推荐顺序会随用户表现变化
- [x] 无 L1 时仍可基于历史正确率推荐
- [x] 无训练历史时可降级为 L1-aware 或普通主题推荐
- [x] localStorage 不可用时训练流程不受影响
- [x] `npx tsc --noEmit` 通过
- [x] `npm run build` 通过

## 影响文件

- `src/types.ts`
- `src/utils/storage.ts`
- `src/utils/recommendation.ts`（新增）
- `src/components/SmartRecommend.tsx`
- `src/components/SessionResultView.tsx`
- `src/App.tsx`

## 约束

- 本阶段仍不引入账号或数据库
- 推荐解释要区分“来自 L1 难点”和“来自你的训练历史”
