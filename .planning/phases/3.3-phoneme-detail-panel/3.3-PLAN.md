# Phase 3.3: Phoneme Detail Panel

## 目标

补齐音素/拼音单元详情面板，让用户能从训练主题、L1-aware 推荐、错题复盘和最小对立体复盘中理解“这个声音为什么难、它像什么、可以用哪些例词继续练”。

## 任务

- [x] 3.3.1 定义音素详情读模型：profile 音素定义、L1 难点、例词、minimal pairs
- [x] 3.3.2 新增音素详情面板组件，展示 PAM/SLM 原因、L1 等价音、例词和最小对立体入口
- [x] 3.3.3 从 Topic 选择区打开当前主题详情
- [x] 3.3.4 从 SmartRecommend 打开详情，并保留一键开始专项训练
- [x] 3.3.5 从拼写结果页的 diff 打开 expected/actual 音素详情
- [x] 3.3.6 从 minimal pair 当前题和复盘打开 target/contrast 详情
- [x] 3.3.7 更新文档和 QA，总结 Phase 3.3

## 退出条件

- [x] 用户可点击任一可训练音素/声调/韵母查看详情
- [x] 有 L1 映射时显示 PAM/SLM 原因、难度等级和 L1 等价音
- [x] 没有 L1 映射时仍显示 profile 标签、类别、例词和可用 minimal pairs
- [x] 详情面板可直接启动该音素的拼写训练或最小对立体听辨
- [x] 空例词、空 minimal pair 和 localStorage 失败不影响训练流程
- [x] `npm run validate:data` 通过
- [x] `npm run lint` 通过
- [x] `npm run build` 通过

## 约束

- 组件仍通过 profile 和工具服务读取语言行为，不直接依赖 `src/data/*`
- 训练器不能依赖 L1 才能打开详情；L1 只增强解释内容
- 本阶段不引入后端、账号、ASR 或标准音频服务
