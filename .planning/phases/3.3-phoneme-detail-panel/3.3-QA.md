## 手动冒烟清单

| 场景 | 预期 |
|---|---|
| Topic 选择具体音素后点击详情按钮 | 打开详情抽屉，显示符号、类别、标签、例词 |
| 有 L1 映射的推荐项点击详情按钮 | 显示难度星级、PAM/SLM 原因、L1 等价音 |
| 无 L1 或无映射时打开详情 | 不阻塞训练，显示可降级说明、例词和可用 minimal pairs |
| 拼写作答错误后点击 diff 里的 expected/actual | 打开对应音素详情 |
| minimal pair 当前题点击 target/contrast 标签 | 打开对应音素详情 |
| minimal pair 结果复盘点击 target/contrast 标签 | 打开对应音素详情 |
| 详情面板点击 Practice | 关闭面板并进入该音素拼写训练 |
| 详情面板点击 Listen | 有材料时关闭面板并进入该音素 minimal pair 听辨；无材料时按钮禁用 |
| 点击遮罩或关闭按钮 | 关闭详情面板，不改变当前训练 session |

## 已知限制

- 详情入口的交互仍缺少组件测试和 E2E 覆盖。
- `PhonemeDetail` 例词来自现有词库，不能证明每条音标或拼音标注的语言学质量。
- Minimal pair 播放仍依赖 Web Speech API；标准音频仍留待后续阶段。
