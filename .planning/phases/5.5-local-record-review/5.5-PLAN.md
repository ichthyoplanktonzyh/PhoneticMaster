# Phase 5.5: Local Record & Review

## 目标

提供纯浏览器、本地优先的录音与回放活动，可被课程引用，也可独立用于自由跟读练习。

## 任务

- [ ] 5.5.1 定义录音身份、元数据和 repository 接口
- [ ] 5.5.2 实现 MediaRecorder 能力检测、录制、停止和资源清理
- [ ] 5.5.3 实现 IndexedDB 保存、读取、播放和删除 adapter
- [ ] 5.5.4 实现可被 Lesson 和自由训练复用的 RecordingActivity
- [ ] 5.5.5 提供隐私说明、权限拒绝与存储失败反馈
- [ ] 5.5.6 添加 repository、降级路径和组件交互测试

## 退出条件

- [ ] 支持的浏览器可录制、回放和删除单条本地录音
- [ ] 同一个录音活动可由课程或自由训练入口调用
- [ ] 页面离开或录音停止后麦克风轨道被释放
- [ ] 权限拒绝、IndexedDB 失败或 MediaRecorder 缺失时其他学习和训练仍可完成
- [ ] UI 明确说明录音不会上传且不提供自动评分
- [ ] `npm run lint`、`npm run test` 和 `npm run build` 通过

## 影响文件

## 非目标

- 云端上传或跨设备同步
- 波形编辑
- ASR、Whisper 或 wav2vec2 发音评分
