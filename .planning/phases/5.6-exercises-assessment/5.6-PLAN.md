# Phase 5.6: Data-driven Exercises & Course Assessment

## 目标

增加两种服务课程目标的声音感知练习，并建立基于现有训练证据的课程目标评价与复习建议。

## 任务

- [ ] 5.6.1 盘点现有 Training/MinimalPair result 可提供的目标证据
- [ ] 5.6.2 定义 ExerciseDefinition、ExerciseQuestion、ExerciseSession 和 result read model
- [ ] 5.6.3 实现共享题目调度、答案记录、计分和结果汇总工具
- [ ] 5.6.4 根据试点课程目标实现 Sounds and Spelling 或等价练习
- [ ] 5.6.5 实现 Odd One Out 或第二种互补感知练习
- [ ] 5.6.6 实现 AssessmentCriterion evaluator，聚合兼容 TrainingEvidence
- [ ] 5.6.7 在模块/课时结果中展示目标达标、证据和复习建议
- [ ] 5.6.8 扩展数据校验、单元测试、组件测试和手动 QA

## 退出条件

- [ ] 至少两种新练习使用同一 session/result 基础能力
- [ ] 每种新练习明确服务至少一个试点 LearningObjective
- [ ] 评价复用现有结果或新练习原生结果，不复制拼写/minimal-pair 判定
- [ ] 未达标只产生解释和建议，不阻止自由训练或其他课时访问
- [ ] 结果页区分单次练习成绩与目标达标判断
- [ ] `npm run validate:data`、`npm run lint`、`npm run test` 和 `npm run build` 通过
