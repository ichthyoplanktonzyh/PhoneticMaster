# Phase 5.3 上下文

## 来源

课程试点需要比 TTS 和词库条目更可信的学习材料。现有普通训练与 minimal pair 主要依赖 Web Speech API，Phase 3.1 QA 已确认其对精细听辨只能“勉强可用”。

Phase 5.1 已负责定义来源与许可准入契约；本阶段实现内容查询、音频解析和小规模可信资源接入，不再补救来源不明的既有素材。

## 关键决策

- 播放顺序为：许可明确的标准音频 → profile TTS → 可恢复的无音频状态。
- 标准音频、解释、例词和口音变体都是 LearningContent，不属于 Training Core。
- 口音是语言自定义的可选媒体维度，不硬编码为 British/American。
- 首批媒体只服务课程试点和高价值 minimal pairs；未覆盖内容继续使用 TTS。
- 候选数据与媒体组合为：CMUdict 发音数据、自有录音、逐文件验证的 Wikimedia Commons/Lingua Libre 音频；候选来源不等于自动批准，仍须通过 5.1 的生产准入门禁。

## 约束

- 外部内容必须记录来源、许可证、署名要求和可再分发状态。
- CC BY-SA 音频需评估改编和相同方式共享义务；NC/ND 资源默认不进入面向未来商业化的生产包。
- 静态资源体积可控，单个资源失败不能阻断课程或自由训练。
- 中文及未来语言不得被迫采用英语的位置、重音或口音模型。
