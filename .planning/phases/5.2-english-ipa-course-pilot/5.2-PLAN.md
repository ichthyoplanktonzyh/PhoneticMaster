# Phase 5.2: English IPA Course Pilot

## 目标

制作一个小而完整的英语 IPA 课程模块，验证学习目标、内容、现有训练活动和评价标准的端到端一致性。

## 任务

- [ ] 5.2.1 根据课程地图确定试点学习者、先备假设和模块学习结果
- [ ] 5.2.2 设计至少三个 Lesson 及其可观察 LearningObjective
- [ ] 5.2.3 为每课编写原创或许可明确的解释、听觉线索、例词与对比材料
- [ ] 5.2.4 为每课配置现有 spelling/minimal-pair/training activity reference
- [ ] 5.2.5 定义每个目标的 AssessmentCriterion 与可接受 TrainingEvidence
- [ ] 5.2.6 建立课程内容 fixture、来源元数据和数据校验
- [ ] 5.2.7 用内容评审和 dry-run 验证目标—内容—活动—评价对齐

## 退出条件

- [ ] 至少一个模块、三个课时具备完整教学设计
- [ ] 每个目标至少对应一份学习内容、一项训练活动和一个评价标准
- [ ] 课程活动配置只引用 Training Core 稳定接口
- [ ] 用户可跳过讲解直接练习，不影响训练会话成立
- [ ] 所有课程文本和媒体来源可追溯并允许分发
- [ ] `npm run validate:data`、`npm run lint`、`npm run test` 和 `npm run build` 通过

## 影响文件

- `src/types.ts`
- `src/data/courses/*`（新增，最终位置由 5.1 决定）
- `src/profiles/en.ts`
- `scripts/validateData.ts`
- `src/utils/__tests__/*`

## 非目标

- 完整英语 IPA 课程
- 课程 UI 和进度导航
- 新训练模式
- 强制先修或锁课
