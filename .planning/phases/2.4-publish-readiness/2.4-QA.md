## 浏览器冒烟

| 场景 | 操作 | 期望 |
|---|---|---|
| 首次进入 | 清空 localStorage 后打开首页 | 出现 L2 必选、L1 可选的 onboarding |
| 无 L1 训练 | 只选 English 或 中文 | 进入训练器，不显示推荐侧栏阻塞 |
| 英语拼写 | English / Basic / Spelling，提交 1 题 | 显示 correct/near/incorrect 与 diff |
| 中文拼音 | 中文 / Basic / Spelling，提交 tone-number 拼音 | 可判定，并显示拼音单元 diff |
| 看词听音 | 切换 Training，按 Space 播放 | 可翻页并完成 session |
| 结果页 | 完成一轮 | 显示得分、Review、Recent Sessions |
| 历史清除 | 点击 Clear | 最近记录清空，训练不受影响 |
| TTS 异常提示 | 使用不支持或无 voice 的浏览器 | 显示非阻塞诊断提示 |

## 已知非阻塞提示

- `npm run build` 仍可能显示 Vite chunk-size warning，当前不阻塞 M2 发布。

- Web Speech API 发音质量依赖浏览器和系统语音，最小对立体阶段会继续标注该限制。
