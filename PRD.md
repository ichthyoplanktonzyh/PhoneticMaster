# IPA Spelling Master — 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
IPA Spelling Master 是一款面向美式英语学习者的国际音标(IPA)拼写训练工具。用户通过听取单词发音，在交互式音标键盘的辅助下完成 IPA 转写，从而提升语音辨识和音标掌握能力。

### 1.2 核心价值
- **听音辨音**：通过真实美式发音训练，建立语音与音标符号的直接映射
- **降低门槛**：内置 IPA 键盘，无需记忆或安装第三方输入法即可输入全部音标符号
- **即时反馈**：实时判断用户转写的正确性，给出正确答案对比
- **AI 驱动**：借助 Gemini 大模型动态生成训练语料，内容永不重复

### 1.3 产品愿景
成为英语学习者掌握美式音标的首选训练工具，覆盖从入门到精通的完整学习路径。

## 4. 功能需求详细描述

### 4.1 AI 生成训练词库 (F1)

**触发时机**：页面首次加载 / 用户点击「Reset Word Set」

**处理流程**：
1. 前端向 `/api/words` 发起 GET 请求
2. 服务端调用 Gemini API，传入 prompt：「Generate 10 random American English words for IPA spelling practice...」
3. Gemini 返回结构化 JSON 数组，每项包含 `word` 和 `ipa` 字段
4. 前端接收并渲染训练列表

**异常处理**：
- API 调用失败 → 前端显示错误日志，不阻塞 UI
- 返回数据非数组格式 → 降级为空数组，用户可点击重置重试

### 4.2 音频播放 (F2)

**触发时机**：用户点击发音按钮（扬声器图标）

**处理流程**：
1. 前端向 `/api/tts` 发起 POST 请求，body 包含当前单词文本
2. 服务端调用 Gemini TTS API (`gemini-3.1-flash-tts-preview`)，使用 `Kore` 语音
3. 返回 Base64 编码的音频数据
4. 前端通过 Web Audio API 解码并播放（采样率 24000Hz）

**状态管理**：
- 播放中：按钮展示音频波形动画，禁止重复点击
- 播放完成：恢复可点击状态

### 4.3 IPA 交互键盘 (F3)

**布局结构**：
| 分区 | 内容 | 数量 |
|------|------|------|
| Vowels (元音) | i, ɪ, eɪ, ɛ, æ, ɑ, ɔ, oʊ, ʊ, u, ʌ, ɚ, ə, aɪ, aʊ, ɔɪ | 16 |
| Consonants (辅音) | p, b, t, d, k, ɡ, f, v, θ, ð, s, z, ʃ, ʒ, h, m, n, ŋ, l, r, j, w, tʃ, dʒ | 24 |
| Marks & Stress (标记与重音) | ˈ (主重音), ˌ (次重音), . (音节分隔), ␣ (空格) | 4 |

**交互规则**：
- 点击任意符号按钮，将其追加至输入框光标位置
- 仅当答题状态为 `neutral`（待判定）时可输入
- 答题后（correct/incorrect）键盘输入被锁定
- 键盘支持展开/折叠，由底部「Hide Keypad / Show Keypad」按钮控制

### 4.4 答案判定 (F4)

**判定逻辑**：
1. 移除用户输入和目标 IPA 两端的 `/` 斜线符号
2. 执行 `trim()` 去掉首尾空格
3. 严格字符串相等比较
4. 相等 → `correct`，得分 +1
5. 不等 → `incorrect`

**状态流转**：
```
neutral → (点击 Check Answer) → correct / incorrect
correct / incorrect → (点击 Next Challenge) → neutral（进入下一题）
```

### 4.5 进度追踪 (F5)

**展示元素**：
- 顶部进度条：以圆点/短线表示每道题的状态（已完成/未完成）
- 当前题目序号：`IPA Practice • {n} / {total}`
- 得分：累计正确数
- 准确率：`score / currentIndex * 100`（四舍五入）

### 4.6 错误提示 (F7)

回答错误时展示：
- 红色横幅：「✗ Correct: /{ipa}/」（显示标准 IPA）
- 题目区域显示：「Actual Word: **{word}**」

### 4.7 重置词库 (F8)

点击页面底部「Reset Word Set」→ 触发 `fetchWords()` 函数 → 清空所有状态 → 重新获取 10 个新单词 → 进度归零

### 4.8 会话结束

完成全部 10 题后弹出 alert：`Quiz Finished! Your score: {score}/10`，随后自动加载新词库。

## 5. 非功能需求

### 5.1 性能
| 指标 | 目标值 |
|------|--------|
| 首屏加载时间 | < 2s |
| API 响应时间（词库生成） | < 5s |
| TTS 音频生成时间 | < 3s |
| 音频播放延迟 | < 500ms |
| 页面交互响应 | < 100ms |

### 5.2 可用性
- 界面适配桌面端（首期不要求移动端响应式）
- 所有可点击元素提供 hover/active 视觉反馈
- 按钮禁用状态有明确视觉区分（opacity: 0.3）

### 5.3 可靠性
- Gemini API 调用失败时，前端展示错误信息但不崩溃
- 输入验证：无输入时「Check Answer」按钮禁用

### 5.4 可维护性
- 前端使用 TypeScript 提供类型安全
- 组件化设计（IPAKeypad 独立组件）
- 服务端路由与静态资源服务分离

### 5.5 安全性
- Gemini API Key 仅存储在服务端环境变量（`.env`），不暴露给前端
- 使用 `dotenv` 加载环境变量
- 生产环境通过环境变量注入，不依赖 `.env` 文件

## 6. 用户界面设计规范

### 6.1 视觉风格
- **配色**：主色 Indigo (#4F46E5)，背景浅灰 (#F8F9FA)，卡片白色
- **字体**：系统无衬线字体，IPA 符号使用 serif 字体
- **圆角**：大圆角设计 (rounded-2xl, rounded-[40px])
- **阴影**：轻阴影 + indigo 色调光晕

### 6.2 布局结构

```
┌──────────────────────────────────────────┐
│  Header: Logo + 进度条 + 分数 + 准确率    │
├──────────────────────────────────────────┤
│                                          │
│          [🔊 发音按钮]                    │
│        Click to hear the word            │
│        (错误时显示实际单词)               │
│                                          │
│     ┌──────────────────────┐             │
│     │   /ɪkˈsprɛs/        │  ← 输入框   │
│     └──────────────────────┘             │
│       (正确/错误反馈横幅)                │
│                                          │
│     ┌──────────────────────┐             │
│     │  IPA 符号键盘        │  ← 可折叠  │
│     │  元音 / 辅音 / 标记  │             │
│     └──────────────────────┘             │
│                                          │
│  [Hide Keypad]     [Clear] [Check ✓]     │
├──────────────────────────────────────────┤
│  [Reset Word Set]     IPA Practice • n/N │
└──────────────────────────────────────────┘
```

### 6.3 交互动效
- 使用 `motion/react` (Framer Motion) 实现动画
- 按钮：whileHover scale(1.05), whileTap scale(0.95)
- 反馈横幅：fadeIn + slideUp 动效
- 键盘：展开/折叠高度动画
- 音频播放：音柱跳动动画

## 7. 技术架构约束

| 层次 | 技术选型 | 说明 |
|------|---------|------|
| 前端框架 | React 19 + TypeScript 5.8 | SPA 架构 |
| 构建工具 | Vite 6 | 开发服务器 + 生产构建 |
| CSS | Tailwind CSS 4 | 原子化 CSS |
| 动效 | Motion (Framer Motion) 12 | 声明式动画 |
| 图标 | Lucide React | SVG 图标库 |
| 后端 | Express 4 + tsx | API 代理 + 静态服务 |
| AI 服务 | Google Gemini API (genai SDK) | 词库生成 + TTS |
| 语音 | Gemini TTS (`Kore` voice) | 美式发音 |

### 架构图
```
Browser ──→ Express Server (port 3000)
                │
                ├── /api/words  ──→ Gemini Flash (text generation)
                ├── /api/tts    ──→ Gemini TTS (audio generation)
                └── /*          ──→ Vite Dev Server (静态资源)
```
